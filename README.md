### Features

- Responsive design (mobile - tablet - desktop).
- 2 languages (english - spanish).
- Animations.
- Lazy load images.
- Suitable images sizes depending on the viewport.
- Preload assets (fonts) and style for fast painting (LCP - FPC).
- Code splitting
